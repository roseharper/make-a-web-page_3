Thinking like a programmer:

  • Procedural Thinking: Giving a computer clear and unambiguous directions.
  • Abstract Thinking: Making similarities among differnt things or ideas.
  • System Thinking: Making a large problem into smaller problems.
  • Technological Empathy: Having an understanding of what a computer can and can't do.

Stage 1: Make your Web Page

The Basics of the Web and HTML

HTML stands for HyperText Markup Language. This is the language used to write a web page.
HTTP stands for HyperText Transfer Protocol. The main protocol of the web.
HTML Tag
  • <NAME>Content</NAME>
Examples of HTML Tag
  • Bold: <b>
  • Italics: <em>
Block creates an invisible box where inline elements do not.
Example of Block Elements
  • <p>
  • <div>
Example of Inline Elements
  • <br>
  • <span>

Creating a Structured Document with HTML

HTML is the "structure" of a webpage.
CSS controls the "style" of a webpage.
JS is the "functionality" of a webpage.
All elements are rectangular.
DOM (Document Object Model) or tree like structure of a page.

Adding CSS Style to HTML Structure

CSS allows programmers to avoid repetition by allowing the user to create "sytle" for more than one section of their web page at one time.
CSS stands for Cascading Style Sheets.
The Box Model

Content is the center box that holds the text and images
Padding goes around the content and is transparent.
Border goes around the padding and content.
Margin goes around the border, padding and content. It is also transparent.

Box Sizing
There are two techniques for box sizing.
  • Set sizes in terms of percents instead of pixels.
  • Set the box-sizing attribute to border-box.
Different browsers work slightly differently. This can lead to displaying your code differently.

Box Positioning
Divs are block elements. By default, they take up the entire width of a page.
Adding the rule display:flex will let divs appear next to each other.
